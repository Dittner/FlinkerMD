`emphasized text`, __bold__, _italic_, ___italicbold___ and ~strong~ element

Begin paragraph with a _tilde mark_ to make a paragraph strong: 
~ The whole paragraph is strong.

To ignore formatting use backslash symbol '\':
\_\_intro.html\_\_

### Emphasizing
It's `<em>` mark tag.
It supports inline formatting:
`__bold__, _italic_ and ___italicbold___ formatting`

__BUT:__ Multiline text ignores `<em>`:
ABC `DE
FG
H`
J

[css]
p {
    color: #c0c1c2;
}

em {
    color: #bfc271;
    font-style: italic;
}

strong {
    color: #d76565;
}
